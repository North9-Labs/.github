Seam — Post-Quantum Transport

User-space UDP transport built in Rust. ML-KEM-768 post-quantum key exchange baked in from day one — not added on top. Designed to replace SSH tunnels and scp with something that's actually safe against quantum computers.

Crypto

  • ML-KEM-768 + X25519 hybrid KEM
  • Noise_XX authenticated handshake
  • ChaCha20-Poly1305 AEAD per packet
  • BLAKE3 key derivation
  • 0-RTT session resumption
  • Header protection (QUIC-style)

Transport

  • Multi-stream over single UDP socket
  • Reliable streams + unreliable datagrams
  • CUBIC / BBRv1 congestion control
  • Forward error correction (Reed-Solomon)
  • Path MTU probing
  • DDoS-resistant stateless cookies
